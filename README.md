### Usage

1. Run each task:

- **Hello World:**
  ```
  node hello-world.js
  ```

- **Server:**
  ```
  node server.js
  ```

- **File System Operations:**
  ```
  node file-system.js
  ```

- **Password Generator:**
  ```
  node password-generator.js
  ```

- **Email Sender:**
  ```
  node email-sender.js
  ```
